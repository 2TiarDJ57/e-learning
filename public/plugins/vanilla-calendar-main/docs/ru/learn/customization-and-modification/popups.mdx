# Поп-апы

Для любого дня в календаре можно добавить поп-ап с информацией, который будет отображаться при наведении на этот день.

В примере ниже мы выделим несколько дней с помощью css-модификаторов и добавим информацию об этом дне.

Подробнее о поп-апах читайте в справочнике, раздел <a href="/docs/reference/additionally/pop-ups">«Поп-апы»</a>
