# DOM-шаблоны

Одним из преимуществ этого календаря является то, что вы можете легко настроить его HTML-разметку с помощью параметра `DOMTemplates`.

Благодаря этому вы можете смело добавлять свою кнопку или любой другой html-элемент в свой календарь.

`DOMTemplates` принимает `type` календаря в качестве ключа. А типы принимают строку в качестве значения.
