# Включить выбор времени

По умолчанию выбор времени отключен.
Включите чтобы установить 12-часовой день или переключите на 24-часовой день если есть такая необходимость.

По мимо этого вы можете установить начальное время при инициализации и управлять шагом минут и часов.

Вариант с 12-часовым днем ​​и маркером AM/PM.

<Sandbox example="additional-features-enable-time-picker-12" height={370} />

Вариант с 24-часовым днем.

<Sandbox example="additional-features-enable-time-picker-24" height={370} />

По умолчанию время устанавливается автоматически. Вы можете это переназначить на свое значение.

Для 24-часового дня маркер AM/PM прописывать не нужно.

<Sandbox example="additional-features-enable-time-picker-your-time" height={370} />

По мимо всего прочего, вы можете управлять шагом ползунков времени. А так же отключить возможность вписывать время вручную в input.
