# Выбор даты в input

Итак, если у вас есть необходимость показывать календарь по клику на **«Input»**, проще всего инициализировать его сразу с параметром <a href="/docs/reference/main/main-settings#input">`input`</a>.
Вот самый простой пример такого использования с настройками по умолчанию.

</Info>

Важно понимать, что **«Input»** в контексте этого календаря не обязательно является тегом `<input>`, это может быть любой тег, например `<div>`. Рассмотрим этот вариант.

<Sandbox example="additional-features-date-picker-in-input-div" height={470} />
