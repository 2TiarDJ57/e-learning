# ISO 8601
**ISO 8601** — это международный стандарт, который описывает форматы дат и времени и предоставляет рекомендации по их использованию в международном контексте.

В контексте этого календаря **ISO 8601** используется для определения первого дня недели. Если `iso8601: true`, первый день недели — понедельник, иначе — воскресенье.
