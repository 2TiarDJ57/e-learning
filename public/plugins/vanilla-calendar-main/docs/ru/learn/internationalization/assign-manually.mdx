# Назначить вручную
Чтобы никого не обделять и не обижать, в календарь добавлена ​​возможность ручной локализации, на случай, если параметр `settings.lang` вас по каким-то причинам не устроит.

Вообще в календаре локализованы только надписи месяцев и недель, поэтому достаточно передать соответствующие массивы строк, а для `settings.lang` установить значение `'define'`.

<Info>
  Обратите внимание, что массив недель должен начинаться с воскресенья, это не влияет на **ISO 8601**.
</Info>
