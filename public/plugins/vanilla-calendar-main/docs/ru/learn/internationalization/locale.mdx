# Локаль
Если ваша локаль поддерживается методом <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Date/toLocaleString" rel="noreferrer" target="_blank">`.toLocaleString()`</a> достаточно передать ее в `settings.lang`.

<Sandbox example="internationalization-locale" />

Если локаль не поддерживается или переведена неправильно, вы всегда можете установить локаль вручную, см. следующий раздел.
