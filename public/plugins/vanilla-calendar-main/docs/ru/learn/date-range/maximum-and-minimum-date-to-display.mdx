# Максимальная и минимальная дата для отображения

Параметры `settings.range.min` и `settings.range.max`, точно так же, как параметры `date.min` и `date.max`, указывают на диапазон дат в календаре, но с одним отличием.

Они **не влияют на жизненный цикл календаря**, а просто указывают, какие даты в диапазоне разрешено отображать и/или выбирать.

Например, если для параметра `settings.visibility.disabled` установлено значение `true`, то минимальный и максимальный год, который может видеть пользователь, будет взят из параметров `date.min` и `date.max`.
