

# Несколько

Тип календаря `'multiple'` отображает несколько месяцев, позволяет выбирать дни, переключаться между месяцами с помощью стрелок, но отключает возможность выбора месяца и года из соответствующих заголовков.

<Sandbox example="type-of-calendar-multiple" vertically={false} height={620} />

Этот тип календаря чаще всего используется в сочетании с параметром `settings.selection.day: 'multiple-ranged'`, который позволяет вам выбрать диапазон дат.
Также давайте отключим прошедшие дни и скроем дни из других месяцев.
