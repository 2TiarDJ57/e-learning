# Параметры только для чтения

После инициализации календаря создаются дополнительные параметры, доступные только для чтения.

Эти параметры содержат набор данных на текущее состояние календаря.

<Info>
  Состояние календаря меняется каждый раз, когда вы взаимодействуете с календарем.
  Соответственно, набор данных будет перезаписан, поэтому эти свойства доступны только для чтения.
</Info>

## HTMLElement

`Type: HTMLElement`

```js
const calendar = new VanillaCalendar('#calendar');
console.log(calendar.HTMLElement);
```

Корневой html-элемент, в котором был инициализирован календарь.

- - -

## HTMLInputElement

`Type: HTMLElement`

```js
const calendar = new VanillaCalendar('#calendar');
console.log(calendar.HTMLInputElement);
```

Корневой html-элемент, который является полем ввода.

- - -

## currentType

`Type: string`

```js

console.log(calendar.currentType);
```

Текущий тип календаря.

- - -

- - -

## userTime

`Type: boolean`

```js
const calendar = new VanillaCalendar('#calendar');
console.log(calendar.userTime);
```

Указывает, как было установлено время при инициализации календаря. Автоопределение текущего времени (по умолчанию) или использование параметра `settings.selected.time`.

- - -

## correctMonths

`Type: number`

```js

console.log(calendar.correctMonths);
```

Текущее кол-во месяцев для отображения, если тип календаря `'multiple'`.

- - -

## viewYear

`Type: number`

```js

```

Текущий отображаемый год.
