

# Настройки

Ключ верхнего уровня `settings` содержит набор параметров, которые определяют настройки календаря.

## settings.lang

`Type: String`

`Default: 'en'`

`Options: Language label | 'define'`

```js
new VanillaCalendar('#calendar', {
  settings: {
    lang: 'en',
  },
});
```

Языковая локализация календаря. Укажите языковую метку <a href="https://www.iana.org/assignments/language-subtag-registry/language-subtag-registry" target="_blank" rel="nofollow noreferrer">BCP 47</a> или `'define'`,
чтобы назначить свои собственные названия месяца и недели, подробнее см. <a href="/docs/reference/additionally/localization">`locale`</a>.

---

## settings.iso8601

`Type: Boolean`

`Default: true`

`Options: true | false`

```js
new VanillaCalendar('#calendar', {
  settings: {
    iso8601: true,
  },
});
```

Начало недели по международному стандарту ISO 8601. Если установлено значение `'false'`, неделя начинается в воскресенье, в противном случае — в понедельник.

---

## settings.range.min

`Type: String`

`Default: '1970-01-01'`

`Options: 'YYYY-MM-DD'`

```js
new VanillaCalendar('#calendar', {
  settings: {
    range: {
      min: '2022-07-01',
    }
  },
});
```

Минимальная дата, которую может выбрать пользователь. Даты меньше указанной, будут считаться отключенными и недоступными для выбора.

<Info>
  Важно знать, что `settings.range.min` и `settings.range.max` отключают даты, а `date.min` и `date.max` вообще их не создают.
</Info>

---

## settings.range.disablePast

`Type: Boolean`

`Default: false`

`Options: true | false`

```js
new VanillaCalendar('#calendar', {
  settings: {
    range: {
      disablePast: true,
    }
  },
});
```

Отключает все прошедшие дни.

---

## settings.range.disableGaps

`Type: Boolean`

`Default: false`

`Options: true | false`

```js
new VanillaCalendar('#calendar', {
  settings: {
    range: {
      disableGaps: true,
    }
  },
});
```

Отключает выбор дней в диапазоне с отключенными датами. Работает, только если для параметра <a href="/docs/reference/additionally/settings#selection-day">`settings.selection.day`</a> установлено значение `'multiple-ranged'`.

---

## settings.range.disableAllDays

`Type: Boolean`

`Default: false`

`Options: true | false`

```js
new VanillaCalendar('#calendar', {
  settings: {
    range: {
      disableAllDays: true,
    }
  },
});
```

Отключает все дни, полезно при использовании <a href="/docs/reference/additionally/settings#range-enabled">`settings.range.enabled`</a>.

---

## settings.range.disableWeekday

`Type: Number[]`

`Default: []`

`Options: number[] | []`

```js
new VanillaCalendar('#calendar', {
  settings: {
    range: {
      disableWeekday: [0, 6],
    }
  },
});
```

Отключить указанные дни в неделе. Укажите массив с типом `number`, где `number` это `id` дня недели. `0` - воскресенье.

---

## settings.range.disabled

`Type: String[]`

`Default: null`

`Options: ['YYYY-MM-DD'] | null`

```js
new VanillaCalendar('#calendar', {
  settings: {
    range: {
      disabled: ['2022-08-10:2022-08-15', '2022-08-20'],
    }
  },
});
```

Отключить указанные даты независимо от указанного диапазона.

<Info>
  Чтобы указать диапазон отключенных дат, используйте любой разделитель между датами в приделах одной строки.
</Info>

---

```js
new VanillaCalendar('#calendar', {
```

## settings.visibility.theme

`Type: String`

`Default: 'system'`

`Options: 'light' | 'dark' | 'system'`

```js
new VanillaCalendar('#calendar', {
  settings: {
    visibility: {
      theme: 'system',
    },
  },
});
```

Тема календаря. По умолчанию тема определяется системой пользователя или настройками сайта.

---

## settings.visibility.themeDetect

`Type: String | False`

`Default: 'html[data-theme]'`

`Options: 'string | false`

```js
new VanillaCalendar('#calendar', {
  settings: {
    visibility: {
      themeDetect: 'html[data-theme]',
    },
  },
});
```
Чтобы календарь отслеживал и применял тему сайта автоматически, вам нужно передать строковое значение в виде CSS-селектора. Где квадратные скобки указывают на атрибут, который содержит название темы.

По умолчанию отслеживается тег `html` с атрибутом `data-theme`, но это может быть любой другой атрибут и тег, например, `class`, если в качестве имени класса используется имя темы: `'html[class]'`.

Если установлено значение `false`, тема определяется системой пользователя или параметром `settings.visibility.theme`.
